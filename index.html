E-Legal



Built with: NodeJS Express MongoDB Football API Web sockets

Project Overview:
A site for users to vote on upcoming EPL Football matches. Users were awarded tokens for every correct prediction. In case users run out of tokens, a cron job would run every sunday night, awarding 10 tokens to all users.
The match info were pulled from an open-source external API. Changes in the match info were updated on the client using web-sockets so users would not need to refresh the page to get the latest updates.
